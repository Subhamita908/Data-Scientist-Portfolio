# 🚀 How To Run Projects

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Data-Scientist-Portfolio.git
```

## Navigate Into Project

```bash
cd Data-Scientist-Portfolio
```

## Install Dependencies

```bash
npm install
```

or for Python projects:

```bash
pip install -r requirements.txt
```
